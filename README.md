# Reservation
PHP, MySQL, Login example

# Installation
* Create database with name 'reservation' or any
* Edit `php/config.php` file as your mysql system 
* Run mysql query from `reservation-db.sql`

# Tested Environment
* Web Server : Apache
* PHP version : 7.1.8
* MySQL version : 5.6.35
